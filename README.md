# Animal-Color-Path
Sex, Population and Species classifier for Salamandrinas and Turtles.
This repository will be updated with a full tutorial onece the paper is published on \hyperref{Journal of Zoology}{https://zslpublications.onlinelibrary.wiley.com/journal/14697998}.
Yet, you can find the datasets and matlab CNN models on \hyperref{huggingface}{https://huggingface.co/MANDRACHE-Lab}.
